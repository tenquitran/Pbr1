Pbr1  - simple OpenGL PBR demo.
Based on https://learnopengl.com/PBR/Lighting

Uses libraries:
	GLEW, version 2.0.0 (http://glew.sourceforge.net/)
	GLM, version 0.9.8.4 (http://glm.g-truc.net/)
	stb_image (http://nothings.org/stb)

Textures: https://www.sketchuptextureclub.com/
